# 🖼️ CodeAlpha Image Gallery

A responsive and modern Image Gallery built using **HTML, CSS, and JavaScript** for the **CodeAlpha Frontend Development Internship**.

## 🚀 Features

- Responsive Grid Layout
- Category Filters
- Lightbox View
- Next / Previous Navigation
- Keyboard Support
- Hover Zoom Effect
- Smooth Animations
- Mobile Friendly Design
- Modern Dark UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
CodeAlpha_ImageGallery/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```
